# OpenLakeTx

OpenLakeTx is a portfolio-grade, open-source lakehouse project built on Google Cloud Platform.

## Vision
Build a Delta-style transactional data lake with:
- ACID guarantees
- Schema evolution
- Bronze / Silver / Gold layers
- Batch and streaming ingestion

## Tech Stack
- Python
- Google Cloud Storage
- BigQuery
- (Future) Apache Spark
